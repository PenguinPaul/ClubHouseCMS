The Happenings

Today we had guest speakers from Microsoft, David Washington and Chris Koenig, talk about Windows Mobile App development. During our club meeting, David showed us how easy it was to create a simple app using AppStudio; an online app creation tool that grabs its content from RSS feeds, YouTube channels, or your own data files. Later in the evening, David and Chris ran a development workshop. There were a ton of great app ideas and A TON of people got their apps created and submitted to the Windows App Store. Now for the icing on top, anyone whose app gets accepted into the Windows Store gets a free Windows Phone! If you're still interested in developing an app for the free phone, feel free to contact me or David.

3D Printing

The 3D Printer has to be one of the coolest things I've used in a while. Being able to see a 3D model go from computer model to physical object is fascinating. Now you can try it out too. If you're interested in learning how to use the 3D printer, schedule a time with Chris. Unless you're granted special permission, you have to go through a short training with Chris before using the printer. Once you're using the printer, make sure you follow the 3D Printer Rules found on the whiteboard behind the printer. If something you need to print requires breaking one of the rules, speak to Chris or an E-Board member first.
